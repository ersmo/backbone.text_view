# backbone.text_view

> Backbone text view

## Usage

```js
view = new Backbone.TextView({
  name: 'user_name',
  placeholder: 'Please enter your username'
});
```

## Option

```coffee
defaults:
  name: 'input'
  type: 'text'
  label: ''
  className: 'input-details'
  defaultValue: ''
  placeholder: ''
  disabled: false
  span: ''
```
